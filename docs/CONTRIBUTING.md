# Contributing

Thank you for your interest in contributing to Finance Master.

## Goals
- Keep logic simple and readable
- Maintain DRY structure
- Avoid breaking existing reports

## Setup
```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## Suggestions
- Improve merchant grouping rules
- Enhance PDF formatting
- Add new summary insights

Please avoid committing real financial data.
